portfolio Performance Measures

To assess the performance of a portfolio we will use the following measures:

Return (annualized): \(R_p = mean(DailyReturns) * 252\)

Risk (annuazlied): \(\sigma_p = \sigma(DailyReturns) * \sqrt{(252)}\)

Sharpe Ratio \(Sharpe_p = \frac{\overline{R_p} - \overline{R_f}}{\sigma_p}\), where \(R_f\)- risk free rate.

Sortino Ratio \(Sortino_p = \frac{\overline{R_p} - \overline{R_f}}{\sigma_d}\), where \(\sigma_d\)- standard deviation of negative portfolio returns.

Beta: \(\beta\) coefficient of the regression of the form \(R_p = R_f + \beta(R_M - R_f)\) , where \(R_M\) is a broad market return (we will use RusselL 2000 as a proxy)

Treynor Ratio: \(Treynor_p = \frac{\overline{R_p} - \overline{R_f}}{\beta}\)
